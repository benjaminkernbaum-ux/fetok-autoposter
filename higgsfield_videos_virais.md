# 🎬 FéTok — Higgsfield AI Production Bible

> Stack completo para produzir os **3 roteiros cinematográficos virais** no Higgsfield AI + ElevenLabs + CapCut.
> Formato: vertical 9:16, 35–45s, otimizado Reels/TikTok/Shorts.

---

## 🧰 Stack de produção (pipeline oficial)

| Etapa | Ferramenta | Output |
|-------|-----------|--------|
| 1. Storyboard / imagens base | Midjourney v7 ou Flux 1.1 Pro | 8–12 keyframes PNG (9:16, 1080×1920) |
| 2. Animação cinematográfica | **Higgsfield DoP I2V** (Lite/Turbo/Standard) | Clipes MP4 5s cada |
| 3. Consistência de personagem | **Higgsfield Soul ID** | Character lock cross-shots |
| 4. Lip-sync (se narrador em cena) | **Higgsfield Speak** | Áudio sincronizado |
| 5. Voz narração | ElevenLabs v3 (voz PT-BR masculina grave) | WAV 48kHz |
| 6. Trilha + SFX | Epidemic Sound / Artlist | Drops cinematográficos |
| 7. Edição final | CapCut Pro (desktop) | MP4 H.264 9:16 |

---

## 🎥 Higgsfield — presets de câmera que VIRALIZAM (DoP controls)

Esses são os **motion presets nativos** do Higgsfield que entregam look cinematográfico sem esforço. Sempre combine 1 movimento principal + 1 secundário.

**Top 10 para conteúdo bíblico:**

1. `Bullet Time` — congela tempo, orbita 360° (clímax espiritual)
2. `Crash Zoom In` — aceleração violenta no rosto (impacto emocional)
3. `Super Dolly Out` — revela escala épica (vale, multidão, deserto)
4. `Vertigo / Dolly Zoom` — tensão psicológica (dúvida, medo)
5. `Robo Arm` — movimento impossível 3D (presença divina)
6. `FPV Drone` — mergulho dinâmico (descida do céu)
7. `Snorricam` — câmera presa no corpo (jornada interior)
8. `Whip Pan` — chicote lateral (transição dimensional)
9. `Low Angle Push In` — submissão/grandiosidade (Cristo, altar)
10. `God's Eye` — top-down descendente (perspectiva divina)

**Regra de ouro:** nunca mais de 2 movimentos por clipe. O algoritmo recompensa clareza visual.

```
cinematic lighting, volumetric god rays, anamorphic lens flare,
```

## ⚙️ Higgsfield settings de produção

**Modo recomendado:**
- **Lite** (5s, $0.10) — pra testes e variações rápidas
- **Standard** (5s, $0.25) — pra shots finais com maior fidelidade
- **Turbo** — pra hooks e shots com movimento rápido

**Resolution:** 1080×1920 (9:16 vertical nativo)
**FPS:** 24fps (look cinematográfico)
**Motion strength:** 0.7–0.85 (força suficiente sem distorção)
**Seed lock:** sempre fixar seed no primeiro shot aprovado pra consistência

## 🧬 Soul ID — consistência de personagem

Pro pescador, Ezequiel e o pastor aparecerem iguais em múltiplos shots:

1. Upload 3–5 fotos de referência do personagem (geradas no Midjourney com `--cref`)
2. Treine Soul ID no Higgsfield
3. Use o Soul ID como input em todos os shots daquele personagem
4. Ganho: zero drift facial entre cortes

Pra cada roteiro, publique:
- **Versão A:** hook atual
