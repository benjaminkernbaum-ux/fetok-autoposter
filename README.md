# FeTok - Automated TikTok Verse Posting Pipeline

Automated cinematic biblical verse video generation and TikTok posting pipeline. Generates stunning visual content with custom fonts, hero backgrounds, and automated scheduling.

## Features

- Automated biblical verse video generation
- - Custom font rendering with cinematic backgrounds
  - - Hero image asset management
    - - Batch video generation pipeline
      - - Auto-fix and recovery systems
        - - Railway deployment ready (Procfile + nixpacks)
          - - Dashboard for monitoring post status
           
            - ## Tech Stack
           
            - - **Runtime**: Node.js
              - - **Video**: FFmpeg, Canvas API
                - - **Deployment**: Railway (nixpacks)
                  - - **Fonts**: Custom typography system
                   
                    - ## Project Structure
                   
                    - - `src/` - Core source code
                      - - `fonts/` - Custom font assets
                        - - `heroes/` - Hero background images
                          - - `fix_*.js` - Auto-repair scripts
                            - - `patch-dashboard.js` - Dashboard patches
                              - - `guia_21_posts.md` - 21-post content guide
                                - - `Procfile` / `nixpacks.toml` - Deployment config
                                 
                                  - ## Getting Started
                                 
                                  - 1. Clone this repository
                                    2. 2. Run `npm install`
                                       3. 3. Configure your environment variables
                                          4. 4. Run `npm start`
                                            
                                             5. ## Deployment
                                            
                                             6. Deployed on Railway with automatic builds via nixpacks.
